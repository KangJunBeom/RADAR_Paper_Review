결론 (논문 4장) — CPU-GPU 간 메모리 대역폭 제한에도 불구하고,
전체 연산 응답속도는 GPU가 DSP 대비 최소 1.2배 ~ 최대 4.2배 우세함을 확인.
하나의 GPU가 제공하는 다중 프로세서 수를 고려하면 다중빔 다기능레이다 신호처리장치에 GPU 적용이 적합하다고 판단됨.
단, 데이터량이 클수록 MEMCPY가 병목이 되므로 설계 시 Host↔Device 전송 최소화 구조가 필수.
